## App icons (required for PWA install)

Create these **PNG** files in this folder:

- `icon-192.png` (192×192)
- `icon-512.png` (512×512)

Tips:

- Use a square image with padding so it looks good when masked.
- Keep the background simple (works well with `theme_color`).

After adding them, re-run Lighthouse and the “App icons” audits should pass.
